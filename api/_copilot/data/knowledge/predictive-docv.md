# Predictive Document Verification (DocV)

Document-and-selfie verification with liveness, document forensics, injection-attack prevention, and deepfake defense. Used as a step-up when database verification is inconclusive, or as a primary check in regulated/high-risk flows.

## When to lead with it
- Fake-ID or deepfake concerns; document-based fraud rings.
- Step-up verification inside onboarding without wrecking conversion.
- Gating high-risk cohorts (e.g. marketplace sellers, gig workers) with stronger assurance.

## Approved claims
- Instant verification for 98.4% of identities on first attempt.
- 0.92-second response time for 95% of transactions.
- 98% liveness detection; multi-frame capture and injection-attack prevention.

## Integration note (approved)
- Latency profile is compatible with inline signup flows; step-up can be conditioned on risk score so only risky cohorts see the document flow.
