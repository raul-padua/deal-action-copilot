# RiskOS Platform

Socure's decisioning platform: orchestrates identity, fraud, compliance, document, device, and behavior signals into configurable real-time workflows — auto-approve, step-up, manual review, decline — with full auditability.

## Positioning (approved)
- Sell the workflow, not the point product: customers ask "what should I do with this applicant," not "is this document real."
- Every Socure module (Sigma models, Verify, DocV, Watchlist, Device Risk) plugs into RiskOS workflows, so a single integration expands to new use cases without re-platforming.
- Expansion motion: land with one module, grow by adding workflow stages (onboarding → account lifecycle → high-risk moments).

## Approved claims
- Unified decisioning reduces vendor sprawl and reconciliation overhead across fraud/compliance tools.
- Policy thresholds are customer-configurable per segment and jurisdiction.
